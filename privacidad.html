4. Base legal del tratamiento

El tratamiento de datos se realiza bajo las siguientes bases legales:

  • Consentimiento explícito — otorgado por el usuario al registrarse en GP1Finance mediante botón de aceptación en el bot.
  • Ejecución del contrato — necesario para prestar el servicio contratado (suscripción Pro).
  • Interés legítimo — mejora del servicio, prevención de fraude y cumplimiento de obligaciones legales.

5. Almacenamiento y seguridad

Los datos de GP1Finance se almacenan en MongoDB Atlas (nube, servidores en EE.UU. — cumple con cláusulas contractuales estándar). Los datos de las apps móviles se almacenan en Firebase (Google Cloud).

Medidas de seguridad implementadas: acceso restringido por credenciales, autenticación requerida para acceder al cluster de base de datos, variables sensibles almacenadas en entorno del servidor (no en código fuente).

Mejoras en progreso: cifrado de campos sensibles en MongoDB (portfolio, datos de pago), auditoría de permisos MongoDB Atlas.

6. Compartición con terceros

GP1 DevStudio no vende datos personales a terceros. Los datos pueden ser procesados por:

  • MongoDB Atlas — almacenamiento de base de datos
  • Telegram — plataforma de mensajería (datos sujetos a política de Telegram)
  • MercadoPago — procesamiento de pagos
  • Google Firebase / AdMob — autenticación, analytics y publicidad en apps móviles
  • Groq / Meta (LLaMA) — procesamiento de consultas IA (sin datos personales identificables)

7. Retención de datos

Los datos se conservan mientras el usuario mantenga una cuenta activa o hasta que solicite su eliminación. Las referencias de pago se conservan por el tiempo que exigen las obligaciones legales tributarias chilenas.
