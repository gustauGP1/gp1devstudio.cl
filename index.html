Código inteligente para tu día a día.

GP1 DevStudio es un estudio de software independiente con base en Santiago de Chile. Desarrollamos bots con inteligencia artificial, aplicaciones móviles y soluciones web para usuarios y empresas que necesitan herramientas que funcionen de verdad.

Trabajamos con stacks modernos y arquitecturas ligeras que permiten lanzar rápido, iterar y escalar sin incurrir en costos de infraestructura innecesarios.

Cada proyecto parte de un principio simple: si no funciona, no sirve.

MarketPulse Chile

Plataforma de inteligencia de mercado financiero para Chile. Monitorea automáticamente Emol Economía, Diario Financiero y La Tercera Pulso cada hora, clasifica el sentimiento del mercado con NLP en español y entrega alertas en tiempo real vía bot de Telegram. Incluye dashboard interactivo, API REST y sistema de suscripciones con 15 días de prueba gratis. Activo como @marketpulse_chile_bot.
